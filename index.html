CAN Bus Security Testbed

A virtual 3-ECU vehicle network (engine, brake, infotainment) built to run a full Threat Analysis and Risk Assessment against real CAN bus and UDS attack classes.

  • • Conducted a full TARA per ISO/SAE 21434, applying STRIDE and attack-tree analysis to score cross-ECU attack paths
  • • Implemented and validated four attack classes: spoofing, replay, denial-of-service, and UDS security-access exploitation
  • • Designed a real-time intrusion detection system to flag each attack against the undefended network
  • • Built an AUTOSAR SecOC-style message authentication layer (AES-128 CMAC) with replay-attack defence
  • • Verified detection and mitigation coverage via an automated pytest suite

Built to bridge hands-on CAN/UDS validation experience with formal ISO/SAE 21434 risk assessment and defence-layer design.
